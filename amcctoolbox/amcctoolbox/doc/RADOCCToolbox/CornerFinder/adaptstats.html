adaptstats

PURPOSE

ADAPTSTATS gets the local mean and standard deviations for each pixel in the image within a region.

SYNOPSIS

function [mimg,stdv]=adaptstats(img,win)

DESCRIPTION 

 ADAPTSTATS gets the local mean and standard deviations for each pixel in the image within a region.
 
 USAGE:

     [mimg,stdv]=adaptstats(img);
 
 INPUTS:
